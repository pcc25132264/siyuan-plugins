# MindChain

A MindChain page plugin based on @xyflow/react, with Protyle rich-text node editing and built-in templates.

## Features

- **Protyle Integration**: Edit nodes using Siyuan's rich text editor.
- **Templates**: Built-in templates for various thinking models.
- **Interactive**: Drag and drop nodes, connect them with different edge types.

## Usage

1. Create a new MindChain document.
2. Use the toolbar to add nodes.
3. Double click a node to edit its content.
