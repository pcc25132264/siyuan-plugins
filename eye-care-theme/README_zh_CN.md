# 护眼模式主题

本插件为思源笔记提供全局护眼配色叠加层，兼容默认主题与第三方主题。

## 功能

- 全局开关，支持快捷键快速启用/禁用
- 预设配色：淡黄 / 淡绿 / 淡灰
- 支持自定义护眼色值（HEX 或 RGB）与强度调节
- 智能时间模式：根据系统时间自动切换护眼模式
- 配置导入/导出（JSON）
- 使用思源存储进行持久化（可随同步在多设备间生效）

## 安装

1. 将本目录打包为 `.zip`
2. 在思源笔记 → 设置 → 集市 → 插件 → 从本地安装

## 使用

- 快捷键：在命令“切换护眼模式”上设置你喜欢的快捷键（思源快捷键设置）。
- 设置面板：在插件设置中打开“护眼设置”。

## 配置文件格式

导出的文件为 JSON：

- `config`：当前配置
- `schemes`：可选的自定义方案列表（预留扩展）

## 说明

- 插件通过将当前主题变量与护眼色进行混合实现，不会替换或破坏现有主题。
- 切换动画时长固定为 250ms。
